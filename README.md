# 🚀 LifeXP - Gamified Productivity Tool

**LifeXP** es una aplicación web de gestión de tareas que transforma la productividad diaria en una experiencia de juego (RPG). El objetivo es motivar al usuario a completar sus pendientes mediante un sistema de niveles y puntos de experiencia (XP).

✨ **[Ver Demo en Vivo aquí](TU_URL_AQUÍ)**

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de la aplicación.
* **CSS3:** Diseño responsivo con estética "Gamer Dark Mode", animaciones y transiciones suaves.
* **JavaScript (Vanilla):** Lógica del sistema de niveles, manejo del DOM y efectos de sonido.
* **LocalStorage:** Persistencia de datos para que el progreso no se pierda al cerrar el navegador.

---

## 💡 Características Principales

* **Sistema de Progresión:** Gana XP según la dificultad de la tarea (Fácil, Media, Difícil).
* **Subida de Nivel (Level Up):** Algoritmo de dificultad creciente (la meta de XP aumenta un 20% en cada nivel).
* **Feedback Auditivo:** Sonidos integrados para completar tareas y subir de nivel.
* **Persistencia Local:** Los datos se guardan automáticamente en el navegador del usuario.
* **Diseño Moderno:** Interfaz optimizada para el enfoque en el usuario (UX).

---


## 🚀 Instalación y Uso
1. Clona este repositorio.
2. Abre `index.html` en tu navegador favorito.
3. ¡Empieza a subir de nivel tu vida!
